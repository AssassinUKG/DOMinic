# DOMinic

A collection of labs for learning DOM XSS.

Answers will be provided by @gr4y-r0se on X.

### Installation

Run the following: 

```bash
git clone git@github.com:Gr4y-r0se/DOMinic.git
pip3 install -r requirements.txt
python3 app.py
```

Or use the provided Docker container.
